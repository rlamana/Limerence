Limerence UI

/ˈlimərəns/ noun, PSYCHOLOGY

An involuntary interpersonal state that involves intrusive, obsessive, and compulsive thoughts, feelings, and behaviors that are contingent on perceived emotional reciprocation from the object of interest.

Buttons

Based on Colin Garven's submit button and Mary Lou's codrops article and implementation.

Attributes

  • primary
    Boolean
    Identifies the button as the primary action
    false
  • secondary
    Boolean
    Identifies the button as a standard action button
    true
  • progress
    Number
    The number that represents current progress percentage (%)
    0

Methods

  • wait()
    Enables the infinite loading state
  • start()
    Programatically enables the progress state (also enabled when progress attribute is > 0)
  • stop()
    Restores button to normal state

Events

  • done
    Fired when the progress bar is completed or the wait stops.

Input

Attributes

  • type
    String
    Specifies the type of input element to display: 'text', 'password', 'email'
    text
  • value
    String
    Input's value
  • name
    String
    Input's name inside a form

Methods

  • focus()
    Gives focus to the input element
  • blur()
    Removes focus from the input element

Checkbox

Attributes

  • checked
    Boolean
    Gets or sets the state, 'true' is checked and 'false' is unchecked.
    false

Methods

  • toggle()
    Toggles checked state

Events

  • changed
    Fired when the checked value changes
